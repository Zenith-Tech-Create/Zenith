# 🚀 Zenith - Quick Setup Guide

Welcome to Zenith! Getting started is **super easy**.

---

## ⚡ **The Fastest Way** (Recommended)

### For Windows Users
1. **Download** `Setup-Zenith-Windows.bat`
2. **Double-click** it
3. **Done!** Zenith will automatically:
   - Check for Node.js (install if needed)
   - Download Zenith from GitHub
   - Install all dependencies
   - Launch the app

### For macOS Users
1. **Download** `Setup-Zenith-macOS.command`
2. **Double-click** it
3. **Done!** Zenith will automatically:
   - Check for Node.js (install if needed)
   - Download Zenith from GitHub
   - Install all dependencies
   - Launch the app

### For Linux Users
1. **Download** `Setup-Zenith-Linux.sh`
2. **Double-click** it (or run in terminal: `./Setup-Zenith-Linux.sh`)
3. **Done!** Zenith will automatically set up and launch

---

## 📋 **What You Need**

- **Node.js** (v14 or higher) - Download free from https://nodejs.org
- **Internet connection** (to download Zenith from GitHub)
- **50 MB free space** (for installation)

---

## 🔑 **First Time Setup**

1. Run one of the setup scripts above
2. App will launch with a **License Activation window**
3. Enter your license key (you got this in your purchase email)
4. **Done!** Zenith is ready to use

**Don't have a license?** Purchase one at https://zenith-app.com/buy ($15 one-time)

## 🆘 **Troubleshooting**

### "Node.js is not installed"
- Download Node.js from https://nodejs.org (LTS version)
- Install it
- Run the setup script again

### "Failed to download from GitHub"
- Check your internet connection
- Make sure you're not behind a restrictive firewall
- Try again in a few moments

### "Failed to install dependencies"
- Make sure you have at least 100 MB free disk space
- Try running the setup script again
- If it keeps failing, please contact support@zenith-app.com

### "License validation failed"
- Make sure you copied the entire license key correctly
- Check that you didn't add any spaces at the beginning or end
- If you're still having issues, email support@zenith-app.com with your license key

## 📚 **Manual Setup** (Advanced Users)

If you prefer to set up manually:

1. **Download** Zenith from GitHub
2. **Extract** the ZIP file
3. **Open** a terminal in the extracted folder
4. **Run:** `npm install`
5. **Run:** `npm start`
6. **Enter** your license key when prompted
