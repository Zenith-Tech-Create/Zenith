================================================================================
                        🔐 SECURITY OVERVIEW
================================================================================

License Key Protection:
  - Format: Base64 encoded
  - Contents: Email + Timestamp + Random + HMAC Signature
  - Algorithm: HMAC-SHA256
  - Cannot be reverse-engineered without SECRET_KEY
  - Each key is unique and tamper-proof

Code Protection:
  - All JavaScript minified by electron-builder
  - Bundled into native executable
  - Reverse engineering difficult (not impossible, but impractical for $15)
  - electron-builder is industry-standard approach

Backend-Free Design:
  - No server = no backend to compromise
  - License validation happens locally
  - No API keys to steal
  - No database to hack
  - Customer data not stored on server
