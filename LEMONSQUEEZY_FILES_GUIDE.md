**Upload 4 files to Lemonsqueezy:**
1. `Setup-Zenith-Windows.bat` - Windows users

- Simple and user-friendly

✅ **All platforms covered**
- Windows: `.bat` file
- macOS: `.command` file

## 📥 **How to Upload to Lemonsqueezy**

### **Step 1: Go to Your Product**
1. Log in to Lemonsqueezy dashboard
2. Go to **Products** → Your Zenith product
3. Scroll to **Digital Downloads** section

### **Step 2: Upload Each File**
1. Click **Add File**
2. Browse and select `Setup-Zenith-Windows.bat`
3. Click **Upload**
4. Repeat for:
   - `Setup-Zenith.command`
   - `Setup-Zenith.sh`
   - `QUICK-SETUP.md`

### **Step 3: Configure Delivery**
1. Make sure **"Automatically deliver files on purchase"** is enabled
2. All 4 files will be included in purchase confirmation email

## 📞 **Questions?**

- **What about the full Zenith source code?**
  - No need! Setup scripts download it from GitHub
  - Saves space and bandwidth
  - Customers always get latest version

- **Can customers skip the setup script?**
  - Yes, they can manually:
    1. Download Zenith from GitHub
    2. Run `npm install`
    3. Run `npm start`
  - But the setup script is 100x easier!

- **What if GitHub is down?**
  - Setup scripts will fail with clear error
  - Customer can try again later
  - This is rare (GitHub has 99.99% uptime)
